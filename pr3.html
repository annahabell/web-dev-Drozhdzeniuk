The ways to open developer console in Mozilla Firefox and Safari

To open the developer console in Firefox, click on the Firefox Menu in the upper-right-hand corner of the browser and select More Tools > Browser Console.

To open the developer console in Safari, firstly select Safari Preferences (Safari Menu > Preferences), select Advanced, and check the "Show Develop menu in menu bar" box. The Safari developer tools are now available from the Develop menu in the menu bar.

Relative or absolute links

Absolute links are preferred for external resources, sharing specific URLs, and when working with content management systems. Relative links are generally more suitable for internal page navigation, providing flexibility during development, and ensuring easier maintenance and portability. Consider the specific use case and requirements of your website when deciding between absolute and relative links.

Image formats used in web development

JPG (JPEG) is commonly used for photographs and complex images, GIF is suitable for animations and simple graphics, PNG offers a balance between quality and file size with transparency support, WEBP provides better compression and smaller file sizes, and SVG is used for scalable vector graphics. The choice depends on factors like image content, quality, file size, and browser compatibility.
